+ Admin Generator

+ User Management

+ CRUD Generator

+ Menu Generator

+ Bulk Insert

+ Bulk Delete

+ Auto Complete

+ Date(time) Picker

+ Soft Deletable

+ Timestampable

+ Filter and Sorting

+ Easy To Customize
